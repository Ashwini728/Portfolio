My Portfolio

GPS Toll based System using Python

This project leverages GPS technology to automate toll collection for vehicles. The system uses Python for programming the backend processes, enabling seamless integration of location data and payment automation.

MDPS

The Multiple Disease Prediction System (MDPS) is a web-based application that leverages machine learning to predict potential diseases based on user-provided health data. This system aims to assist users in early detection of health risks by analyzing symptoms and other health indicators.

Todo App Website Using React

This website is a feature-rich Todo App built using React, aimed at streamlining task management. It provides a clean and responsive interface where users can easily add, edit, and delete their tasks.

Drawbox

DrawBox is a modern web application designed to simplify the process of organizing and managing random participant selection for lucky draws and lotteries. Whether for small community events or large-scale corporate promotions, DrawBox offers an efficient, user-friendly, and transparent platform to automate random selections, eliminating the inefficiencies of traditional methods.

Github Profile Fetch

This web application allows users to fetch and display the profile and bio information of any GitHub user by entering their GitHub username. The tool makes use of GitHub's public API to retrieve details such as the user's name, bio, repositories, followers, and more. With an intuitive interface, users can quickly access a GitHub user's profile data without navigating through GitHub's interface.

Calculator

This is a simple and intuitive web-based calculator designed to perform basic arithmetic operations. Developed using HTML, CSS, and JavaScript, the calculator allows users to add, subtract, multiply, and divide numbers with ease.

Joblisting Webpage using React

This web application allows users to explore job opportunities in various industries through an intuitive and responsive interface. Built with React, the platform provides real-time job listings and enables users to filter jobs based on location, job type, and other criteria.

Facial Expression Recognition using ML

This project implements a Facial Expression Recognition (FER) system using Convolutional Neural Networks (CNNs) with TensorFlow and Keras. The goal is to classify facial expressions into seven categories, enabling automated analysis of human emotions from images.
